Acerca del Proyecto

Este proyecto escolar forma parte de la práctica "Introducción a las Aplicaciones Web" y tiene como objetivo mostrar la evolución de los objetos cotidianos a lo largo del siglo XX.

Objetivos del Proyecto

  • Crear una aplicación web funcional con HTML, CSS y JavaScript
  • Implementar una base de datos para almacenar información
  • Desarrollar un backend con Node.js y Express
  • Mostrar la evolución tecnológica y de diseño del siglo XX

Tecnologías Utilizadas

HTML5 CSS3 JavaScript Node.js MySQL
